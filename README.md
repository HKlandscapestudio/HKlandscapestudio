# Hoàng Kim Landscape Studio – Website V2.0

Website một trang bằng HTML, CSS và JavaScript thuần, phù hợp GitHub Pages.

## Chạy thử
Mở `index.html` hoặc dùng VS Code + Live Server.

## Cập nhật thông tin
- Thông tin thương hiệu, liên hệ: `data/config.js`
- Dịch vụ: `data/services.js`
- Portfolio: `data/portfolio.js`
- Dự án: `data/projects.js`
- Green365: `data/green365.js`

## Tên ảnh cần dùng
- Logo chính: `assets/logo/logo-hoang-kim-landscape-studio.png`
- Logo nền tối: `assets/logo/logo-hoang-kim-landscape-studio-light.png`
- Favicon: `assets/logo/favicon.png`
- Ảnh Hero: `assets/hero/hero-landscape.jpg`
- QR Zalo: `assets/contact/qr-zalo.png`

Ảnh card phải đặt đúng đường dẫn đã khai báo trong các file thuộc thư mục `data`.
Nếu ảnh chưa tồn tại, website tự hiển thị ô hướng dẫn đường dẫn ảnh cần thêm.

## Đăng lên GitHub Pages
1. Tạo repository tên `hklandscapestudio.github.io`.
2. Upload toàn bộ nội dung bên trong thư mục này vào nhánh `main`.
3. Vào Settings → Pages → chọn Deploy from a branch → `main` / root.
